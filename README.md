Student Report Card System
================

📌 Overview
================
The Student Report Card System is a Python-based project that demonstrates Object-Oriented Programming (OOP) concepts such as Classes, Objects, Constructors, Methods, and Inheritance.

The system allows users to:

Add student details
Add or update grades
Display a student's report card
Calculate total marks and average score

🚀 Features
================
Create student records
Store subject-wise marks
Update existing grades
Generate report cards
Calculate total and average marks
Demonstrate inheritance using a SchoolStudent class

🛠️ Technologies Used
================
Python 3
Object-Oriented Programming (OOP)

📖 OOP Concepts Implemented
================
1. Class
A blueprint used to create Student objects.

2. Object
Instances of the Student class.

3. Constructor
The __init__() method initializes student details.

4. Methods
Functions inside the class used to perform operations such as adding grades and displaying report cards.

5. Inheritance
The SchoolStudent class inherits properties and methods from the Student class.


▶️ How to Run
================
Clone the repository:
git clone <repository-url>
Navigate to the project folder:
cd student_report_card
Run the Python file:
python student_report_card.py
